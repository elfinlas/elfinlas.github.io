우분투 업데이트?

간만에 집에서 개발용으로 쓰는 서버 컴퓨터에 콘솔로 접근하니 아래와 같은 메세지가 뜨더군요.

저번 업데이트 이후 오랜만에 켜서 그런지 업데이트가 좀 있더라구요.
예전에 이걸 어떻게 업데이트 하는지 몰라서 방치하고 있었는데 이번 포스팅에서 업데이트 방법을 기록해두려 합니다.

sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade

위 명령어를 차례대로 수행하면 됩니다.
