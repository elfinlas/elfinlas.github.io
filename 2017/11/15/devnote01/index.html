오늘의 삽질

요 몇일 회사업무 +개인 일정으로 인해 바빠서 포스팅을 제대로 못했네요.

사실 바쁘다는것은 핑계인듯 하고…깊게 반성하는 부분입니다…

그래서 오늘의 연구일지는? (사실 이 글은 어제 퇴근하며 작성한 일지입니다.)


오늘도 안드로이드와 혈투(?)

지금 만들어 가는 것은 서버에 Rest api를 호출하여 가지고 온 값을 앱에서 보여주는 작은 서비스(?)를 개발하고 있습니다.

오늘의 삽질은 안드로이드 영역입니다.

안드로이드 영역에서 그래프를 구현해야 하는데 잘 구현되어 있는 것을 찾아보다가
를 알게 되어 구현을 하였습니다..

MPAndroidChart

이 그래프 라이브러리를 쓰다 보니 정말 잘 구현되어 있었습니다.

문제는 웹에서는 D3 라이브러리를 써서 쉽게 구현 하였지만,
저 라이브러리에서 데이터 표시를 동일하게 하는 것은 어려웠습니다.

사실 그래프 그리는 것은 그대로이지만, 표현을 하는 방식을 동일하게 하는 것은 어려웠습니다.

일단 보여지는 데이터는 어느정도 맞춰서 타협을 하였지만, 기회가 된다면 간단한 bar 그래프를 그리는 라이브러리를 하나 만들어보고 싶다는 생각이 들었습니다.

라고 하지만 저는 웹 백엔드 영역을 좀 깊게 파보고 싶을 뿐이고…


ps : 출퇴근 1시간 30분은 너무 고통스럽네요..
