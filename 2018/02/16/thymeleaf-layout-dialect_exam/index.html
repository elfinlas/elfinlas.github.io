JSPл§Ң м“°лӢӨк°ҖвҖҰ

мӮ¬мӢӨ м ҖлҠ” Spring bootлҘј м“ҙм§ҖлҠ” м–јл§Ҳ лҗҳм§Җ м•Ҡм•ҳмҠөлӢҲлӢӨ.
к·ё м „м—җлҠ” Spring + Jsp + mybatis мЎ°н•©мқ„ л§Һмқҙ мӮ¬мҡ©н•ҳмҳҖм§Җмҡ”.
Spring BootлҘј м ‘н•ҳкІҢ лҗң кІғмқҖ 4мӣ”лӢ¬ мҜ”мқҢ нҶ мқҙ н”„лЎңм қнҠёлҘј н•ҳлӮҳ м§„н–үн•ҳл©ҙм„ң м“°кІҢ лҗҳм—ҲмҠөлӢҲлӢӨ.
Springмқ„ м“°лӢӨк°Җ Spring BootлҘј мӮ¬мҡ©н•ҳмҳҖмқ„ л•ҢлҠ” мӢ м„ёкі„ мҳҖмҠөлӢҲлӢӨ.
Spring Bootм—җм„ңлҠ” ViewлӢЁмқ„ JSPліҙлӢӨлҠ” ThymeleafлқјлҠ” кІғмқ„ мӮ¬мҡ©н•ҳкёё к¶ҢмһҘн•ҳкі  мһҲмҠөлӢҲлӢӨ.

ThymeleafлҠ” мҲңмҲҳ HTMLм—җ HTML5л¬ёлІ•мқ„ мӮ¬мҡ©н•ҳм—¬ Server side лЎңм§Ғмқ„ мҲҳн–үн•  мҲҳ мһҲлҠ” HTML нғңк·ё л°Ҹ мҶҚм„ұ кё°л°ҳмқҳ Template Engineмһ…лӢҲлӢӨ.
мқјлӢЁ Spring Boot м—җм„ңлҠ” JSPлҘј м“°л Өл©ҙ м•Ҫк°„ ліөмһЎн•ҙм§‘лӢҲлӢӨ.
л§Ңм•Ҫ Spring Boot м—җм„ң JSPлҘј м“°кёё мӣҗн•ңлӢӨл©ҙ кіөмӢқл¬ём„ң мҳҲм ңлҘј м°ёкі н•ҳм„ёмҡ”.
(лҹ¬лӢқм»ӨлёҢлҸ„ лҶ’мқҖ нҺёлҸ„ м•„лӢҲлқј н•ңлІҲ мҚЁліҙмӢңлҠ” кІғмқ„ м¶”мІң л“ңлҰҪлӢҲлӢӨ :) )

м„ңлЎ мқҖ м—¬кё°к№Ңм§ҖлЎң н•ҳкі вҖҰ
м ңк°Җ ThymeleafлҘј м“°л©ҙм„ң кіөнҶө л Ҳмқҙм•„мӣғм—җ лҢҖн•ң мІҳлҰ¬лҘј н•ң л¶Җл¶„м—җ лҢҖн•ҙ нҸ¬мҠӨнҢ…мқ„ н•ҙліҙкі мһҗ н•©лӢҲлӢӨ.
мӢңмһ‘н•ҳкё° м•һм„ң к°ңл°ңнҷҳкІҪмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • Spring Boot = вҖҳ1.5.8.RELEASEвҖҷ
  • dependencies = compile(вҖҳorg.springframework.boot:spring-boot-starter-thymeleafвҖҷ)


spring-boot-starter-thymeleafм—җлҠ”?

лЁјм Җ мҳҲм ңлҘј мӮҙнҺҙліҙкё° мқҙм „м—җвҖҰ
к°ңл°ңнҷҳкІҪм—җм„ң мқҳмЎҙм„ұм—җ spring-boot-starterм—җ лӢӨм–‘н•ң кІғмқҙ мһҲм§Җл§Ң к·ё мӨ‘ thymeleafлҘј мӮ¬мҡ©н•ҳмҳҖмҠөлӢҲлӢӨ.
мқҙ мқҳмЎҙм„ұмқҖ лӢӨмқҢкіј к°ҷмқҙ кө¬м„ұлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

мқҙ мӨ‘ л§Ё л§Ҳм§Җл§үм—җ м¶”к°Җлҗң Thymeleaf Layout DialectлҠ” нғҖмһ„лҰ¬н”„м—җм„ң мҪ”л“ң мһ¬мӮ¬мҡ©мқ„ к°ңм„ н•ҳкё° мң„н•ҙ мһ¬мӮ¬мҡ©мқҙ к°ҖлҠҘн•ң л Ҳмқҙм•„мӣғкіј н…ңн”ҢлҰҝмқ„ мһ‘м„ұн•  мҲҳ мһҲлҸ„лЎқ лҸ„мҷҖмЈјлҠ” лқјмқҙлёҢлҹ¬лҰ¬мһ…лӢҲлӢӨ.
мқҙкІғм—җ лҢҖн•ң нҸ¬мҠӨнҢ…мқҖ Thymeleafм—җм„ң thymeleaf-layout-dialect мҶҢк°ң нҸ¬мҠӨнҢ…лҘј м°ёкі н•ҙмЈјм„ёмҡ”.


мҳҲм ңм—җ м•һм„ң

мҳҲм ңлҠ” м ңк°Җ мҳӨн”ҲмҶҢмҠӨлЎң л§Ңл“Өкі  мһҲлҠ” CompanyBoardм—җм„ң м“°мқё л¶Җл¶„мһ…лӢҲлӢӨ.
thymeleaf-layout-dialectлҘј мӮ¬мҡ©н•ҳм—¬ кіөнҶөм ҒмңјлЎң мӮ¬мҡ©н•ҳлҠ” л Ҳмқҙм•„мӣғмқ„ л§Ңл“Өкі , м»Ён…җмё  нҺҳмқҙм§Җм—җ кіөнҶө л Ҳмқҙм•„мӣғмқ„ м Ғмҡ©н•ҳлҠ” мҲңм„ңлЎң м§„н–үн•ҳкІ мҠөлӢҲлӢӨ.


1. кіөнҶөм ҒмңјлЎң мӮ¬мҡ©н•  Main Layout мһ‘м„ұ

HTMLмқҙлқјл©ҙ лӢӨмқҢкіј к°ҷмқҖ кё°ліё м–‘мӢқмқ„ л”°лҘј кІғмһ…лӢҲлӢӨ.

html нғңк·ё м•Ҳм—җ headмҷҖ bodyк°Җ кө¬м„ұлҗҳлҠ”лҚ° м Җмқҳ кІҪмҡ° мң„ мҳҲм ңмІҳлҹј нҒ° нӢҖмқҳ кіөнҶө л Ҳмқҙм•„мӣғмқ„ мһЎкі , headмҷҖ body мҳҒм—ӯм—җм„ң th:replace мҶҚм„ұмқ„ нҶөн•ҙ к°Ғ л¶Җл¶„м—җ лҢҖн•ң кіөнҶө л Ҳмқҙм•„мӣғмқ„ мһ¬м •мқҳ н•ҳмҳҖмҠөлӢҲлӢӨ.

th:replace мҶҚм„ұмқҖ к°„лӢЁн•ҳкІҢ м„ӨлӘ…н•ҳл©ҙ н•ҙлӢ№ мҶҚм„ұмқҙ м„ м–ёлҗң html нғңк·ёлҘј replaceм—җ м •мқҳлҗң лӢӨлҘё html нҢҢмқјлЎң м№ҳнҷҳн•ҳлҠ” кІғмңјлЎң jspмқҳ includeмҷҖ л№„мҠ·н•ң лҸҷмһ‘мқ„ н–үн•©лӢҲлӢӨ.

th:replace мҶҚм„ұмқҳ кІҪлЎңлҠ” Spring-bootмқҳ кё°ліё кІҪлЎңмқё templates н•ҳмң„м—җ м •мқҳн•ң кІҪлЎңлҘј мһ‘м„ұ нӣ„ :: л’Өм—җлҠ” м•„лһҳм—җм„ң мҶҢк°ңн•  fragmentмҶҚм„ұмқҳ мқҙлҰ„мқ„ м Ғм–ҙмӨҚлӢҲлӢӨ.


2. Headмқҳ кіөнҶө л¶Җл¶„ мһ‘м„ұ

headмқҳ кіөнҶө л¶Җл¶„мқҖ м•„лһҳмқҳ мҳҲм ңмҷҖ к°ҷмҠөлӢҲлӢӨ.

head нғңк·ём—җм„ң ліҙл©ҙ fragment мҶҚм„ұм—җ мқҙлҰ„мқ„ м •мқҳн•ҳмҳҖкі , th:includeмҶҚм„ұкіј css м„ м–ё, к·ёлҰ¬кі  м»Ён…җмё  нҺҳмқҙм§Җм—җм„ң мӮ¬мҡ©н•  мҳҒм—ӯмқё layout:fragment=вҖқcustom_cssвҖқлҘј м •мқҳн•ҳмҳҖлӢӨ.

th:include=вҖқfragments/common/c_headвҖқмҶҚм„ұмқҳ нҢҢмқјмқҖ лӢӨмқҢкіј к°ҷмқҙ кө¬м„ұлҗҳм–ҙ мһҲлӢӨ.

layout:fragment=вҖқhtml_headвҖқмҶҚм„ұмқҖ м»Ён…җмё  нҺҳмқҙм§Җм—җм„ң titleмқ„ м •мқҳн•ҳлҠ” л¶Җл¶„мңјлЎң мӢӨм ң м»Ён…җмё  нҺҳмқҙм§Җм—җм„ң м•Ңл§һкІҢ мӮ¬мҡ©н•ҳл©°, 4лІҲ м»Ён…җмё  нҺҳмқҙм§Җ лӮҙмҡ©мқ„ м°ёкі н•ҳм„ёмҡ”.
мӨ‘к°„мқҳ add_lib_cssмҳҒм—ӯмқҖ м»Ён…җмё  нҺҳмқҙм§Җм—җм„ң л”°лЎң м¶”к°Җм ҒмңјлЎң мӮ¬мҡ©н•  css л“ұмқ„ м¶”к°Җн•ҳлҠ” мҳҒм—ӯмһ…лӢҲлӢӨ.

к°„лһөнһҲ м •лҰ¬лҘј н•ҳмһҗл©ҙ layout:fragment мҶҚм„ұмқҖ кіөнҶө л Ҳмқҙм•„мӣғм—җм„ң м„ м–ён•ҳм—¬ мӢӨм ң м»Ён…җмё  нҺҳмқҙм§Җм—җм„ң м„ м–ёлҗң лӮҙмҡ©мқ„ мұ„мҡ°кІҢ н•ҙмӨҚлӢҲлӢӨ.


3. Bodyмқҳ кіөнҶө л¶Җл¶„ мһ‘м„ұ

1лІҲм—җм„ң мһ‘м„ұн•ң мҪ”л“ңм—җм„ң body мҳҒм—ӯмқ„ ліҙкІ мҠөлӢҲлӢӨ.

body мҳҒм—ӯмқҖ мӢ«м ң м»Ён…җмё  нҺҳмқҙм§ҖлҘј мұ„мҡё мҳҒм—ӯмқё layout:fragment=вҖқcontent_bodyвҖқмқҙ м„ м–ёлҗҳм–ҙ мһҲкі , кіөнҶөм ҒмңјлЎң мӮ¬мҡ©н•  footerлҘј include н•ҳкІҢ лҗ©лӢҲлӢӨ.

м•„лһҳмқҳ мҳҒм—ӯмқҖ JSлҘј м„ м–ён•ҳм—¬ мӮ¬мҡ©н•ҳлҠ” мҳҒм—ӯмһ…лӢҲлӢӨ.


4. м»Ён…җмё  нҺҳмқҙм§Җ мһ‘м„ұ

м»Ён…җмё  нҺҳмқҙм§ҖлҠ” мң„мқҳ мҳҲм ңлҘј м°ёкі н•ҳмӢңл©ҙ лҗ©лӢҲлӢӨ.
мІҳмқҢ ліҙл©ҙ м•Ҫк°„мқҖ ліөмһЎн•ҳм§Җл§Ң, мң„мқҳ мҳҲм ңлҘј н•ҳлӮҳм”© нқҗлҰ„м—җ л§һм¶°м„ң ліҙкІҢ лҗҳл©ҙ к·ёлҰ¬ м–ҙл өм§ҖлҸ„ м•ҠмҠөлӢҲлӢӨ.
к·ёлҰ¬кі  мқҙн•ҙ нӣ„ мӮ¬мҡ©н•ҳкІҢ лҗҳл©ҙ л Ҳмқҙм•„мӣғм—җ лҢҖн•ң кіөнҶөм Ғмқё л¶Җл¶„мқ„ мүҪкІҢ кө¬нҳ„н•  мҲҳ мһҲмңјл©°, мҪ”л“ң ліөмһЎлҸ„лҸ„ лӮ®м•„м§ҖкІҢ лҗ©лӢҲлӢӨ.


кІ°лЎ 

мң„мқҳ мҳҲм ңлҘј м•Ҫк°„ м •лҰ¬н•ҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • мғҒмң„ кіөнҶө л Ҳмқҙм•„мӣғ -> layout/main_layout.html
  • head кіөнҶө л Ҳмқҙм•„мӣғ -> fragments/main/main_header.html -> fragments/common/c_head.html
  • body кіөнҶө л Ҳмқҙм•„мӣғ -> fragments/main/main_body.html
  • мң„мқҳ кіөнҶө л Ҳмқҙм•„мӣғл“Өмқ„ л¬¶м–ҙм„ң м»Ён…җмё  нҺҳмқҙм§Җ мһ‘м„ұ

нҸ¬мҠӨнҢ…мқҳ лӮҙмҡ©мқҙ м•Ҫк°„ кёёкІҢ лҗҳм–ҙ м ңк°Җ мһҗм„ён•ҳкІҢ м Ғм§Җ лӘ»н•ң л¶Җл¶„лҸ„ мһҲмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.
мқҙн•ҙк°Җ м•ҲлҗҳмӢңкұ°лӮҳ мһҳлӘ»лҗң л¶Җл¶„мқҙ мһҲлӢӨл©ҙ л©”мқј л“ұмңјлЎң м•Ңл ӨмЈјмӢңл©ҙ лӢөліҖ лҳҗлҠ” мҲҳм • мІҳлҰ¬н•ҳлҸ„лЎқ н•ҳкІ мҠөлӢҲлӢӨ.

к°җмӮ¬н•©лӢҲлӢӨ.
