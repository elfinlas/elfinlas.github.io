Pyhwp으로 파일 변환 시 에러가 발생

전에 포스팅했던 Pyhwp 설치에서는 설치 및 사용법만 작성하였다.
그런데 다른 PC에서 진행할 때 이상한 에러를 보게 되었다.

이 경우 아래의 명령어를 수행한다.

pip install OleFileIO_PL==0.23

위 명령어를 수행하다가 lxml 이 없다는 에러가 뜰 경우

pip install lxml
