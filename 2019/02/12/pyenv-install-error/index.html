Pyenv 설치하다가 에러난 경우

작성일 2019-02-12 | Edited on 2019-03-02 | In python |

Pyenv 설치 에러

토이 프로젝트를 Django로 만들어서 진행하였다.
그런데 장비를 여러개를 쓰기에 다른 장비에서 Pyenv를 설정하다 아래와 같은 에러를 만나게 되었다.

위와 같은 에러 메세지가 발생한 경우
아래의 brew 작업을 진행해보자

xcode-select –install //수행
brew install homebrew/dupes/zlib

만약 위 방법을 진행했음에도 에러가 계속 발생하는 경우 아래의 작업을 추가적으로 진행해보자.

brew install homebrew/core/zlib
brew upgrade
brew reinstall zlib
