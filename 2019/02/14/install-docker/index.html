CentOs에 Docker 설치하기

프로젝트 운영을 Aws의 Ec2 CentOS에서 진행을 하는데 테스트를 위해 Docker를 설치해야 했다.
이번 포스팅에서는 이 부분을 정리해보고자 한다.


설치 순서

설치 순서는 아래와 같다.

1. 사전 설치

yum install -y yum-utils device-mapper-persistent-data lvm2

먼저 위 명령어를 통해 사전에 필요한 패키지를 미리 설치한다.

2. Docker 저장소 추가 및 설치

yum-config-manager –add-repo https://download.docker.com/linux/centos/docker-ce.repo
yum install -y docker-ce

처음 명령어는 Docker 최신 버전을 위해 저장소를 추가한 것이고, 아래는 Docker를 설치하는 부분이다.

3. Docker 서비스 시작

systemctl start docker

(기타) Docker 설치 구동 테스트

docker run hello-world

이것은 굳이 안해도 되지만, 혹시 정상적으로 설치가 되었는지 확인해야 할 때 진행해준다.

(기타) Docker 관리자 그룹 추가

sudo groupadd docker
sudo usermod -aG docker $USER
systemctl restart docker

매번 Docker 명령어를 수행할 때 root 권한이 필요하다.
그래서 이 부분을 해결하기 위해 docker를 관리자 그룹에 추가하고 재시작을 해주면 된다.
